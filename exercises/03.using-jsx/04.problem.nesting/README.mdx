# Nesting JSX

<EpicVideo url="https://www.epicreact.dev/workshops/react-fundamentals/nesting-jsx" />

👨‍💼 Remember when we created this with `createElement`?

```html
<div class="container">
	<p>Here's Sam's favorite food:</p>
	<ul class="sams-food">
		<li>Green eggs</li>
		<li>Ham</li>
	</ul>
</div>
```

That was... intellectually stimulating 😅. Well, now try doing the same thing
using JSX. I promise it'll be more fun this time.

<callout-info class="aside">
	💰 Tip: remember `class` in JSX is `className`.
</callout-info>
